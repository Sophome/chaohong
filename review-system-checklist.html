01

测评定位与内容原则

内容定位

  • 面向中国市场消费者，强调购买决策价值。
  • 兼顾科普、实测、推荐、避坑和选购攻略。
  • 用用户听得懂的话解释复杂参数。
  • 避免只复述官方卖点，要输出真实体验判断。

测评原则

  • 测试条件公开，结论可复现。
  • 优点、缺点、适合人群同时呈现。
  • 参数不等于体验，体验也要有数据支撑。
  • 商单与非商单标准一致。

商业原则

  • 明确标注赞助、样机、佣金链接。
  • 不因商单隐藏关键缺点。
  • 推荐逻辑优先于单品转化。
  • 建立长期信任，而不是一次性成交。

通用评分模型

测评维度 权重 核心检查项 输出结论
产品定位与配置 10% 价格段、目标人群、核心配置、同价位竞争力、版本差异。 这款产品到底卖给谁？
设计与做工 10% 外观、材质、结构、接口、易清洁性、安装便利性。 是否好看、好用、耐用？
核心性能 20% 跑分、速度、制冷/制热、清洁能力、画质、音质等品类关键指标。 硬实力是否达标？
真实场景体验 20% 日常使用、重度使用、家庭场景、老人/儿童友好度。 实际用起来爽不爽？
稳定性与可靠性 10% 长时间运行、发热、卡顿、故障率、软件更新、售后反馈。 能不能长期放心用？
能耗 / 噪音 / 安全 10% 功耗、电费、噪音、温控、用电安全、儿童安全。 使用成本和安全性如何？
生态与售后 10% App、智能家居联动、耗材、保修、安装、维修价格。 买完之后省不省心？
价格与购买建议 10% 历史价格、优惠节点、竞品对比、推荐等级、避坑提示。 现在值不值得买？

基础信息采集清单

✓ 产品名称、型号、版本、颜色、容量、规格。
✓ 入手渠道：自购 / 借测 / 品牌送测 / 商单。
✓ 到手价格、官方价、历史低价、促销补贴价。
✓ 生产日期、批次、固件版本、系统版本。
✓ 包装清单、配件完整性、安装服务情况。
✓ 测试环境：温度、湿度、网络、电压、房间面积。
✓ 对照产品：上一代、竞品、同价位热门款。
✓ 测试周期：开箱、短测、长测、复测时间。
05

数码产品专项测评清单

手机 / 平板

性能 影像 续航
  • 屏幕：亮度、色准、护眼、刷新率、户外可视性。
  • 性能：跑分、游戏帧率、温控、后台保活。
  • 影像：白天、夜景、人像、视频防抖、变焦。
  • 续航：轻度、中度、重度使用，充电速度。
  • 系统：广告、流畅度、隐私、生态互联。
  • 购买建议：标准版 / Pro / Max 哪个更值。

笔记本 / 台式机

生产力 游戏 扩展
  • CPU / GPU 性能，单核、多核、生产力软件测试。
  • 散热与噪音：烤机、键盘温度、风扇策略。
  • 屏幕：分辨率、色域、亮度、比例、观感。
  • 接口：雷电、USB、HDMI、网口、读卡器。
  • 续航：办公、视频、移动剪辑场景。
  • 可维护性：内存、硬盘扩展、清灰难度。

耳机 / 音箱

音质 降噪 佩戴
  • 声音：低频、中频、高频、声场、人声表现。
  • 降噪：低频噪音、人声噪音、风噪、通透模式。
  • 通话：室内、街边、地铁、风噪环境。
  • 延迟：游戏、视频、双设备切换。
  • 佩戴：重量、夹头、耳压、长时间舒适度。
  • 续航：单次续航、盒子总续航、快充。

电视 / 投影 / 显示器

画质 亮度 系统
  • 画质：亮度、对比度、色彩、HDR、暗场细节。
  • 动态：运动补偿、拖影、游戏延迟、VRR。
  • 音质：低频、人声、声场、外接音响必要性。
  • 系统：广告、开机速度、会员体系、投屏体验。
  • 安装：桌面 / 壁挂 / 投影距离 / 幕布适配。
  • 使用场景：客厅观影、游戏、办公、副屏。
06

家电产品专项测评清单

冰箱

  • 容量布局：冷藏、冷冻、变温区是否合理。
  • 保鲜能力：蔬菜、水果、肉类、干湿分储。
  • 温控稳定性：不同层温差、开门后恢复速度。
  • 噪音与能耗：夜间噪音、日耗电量。
  • 除味抗菌：串味、净味模块、耗材成本。
  • 适合人群：单人、情侣、三口之家、多代同堂。

洗衣机 / 烘干机

  • 洗净能力：污渍去除、残留、漂洗效果。
  • 护衣能力：磨损、缠绕、缩水、羊毛/真丝模式。
  • 烘干效果：均匀度、褶皱、含水率、绒毛收集。
  • 噪音震动：脱水、烘干、高速运行。
  • 清洁维护：筒自洁、滤网、冷凝器、排水。
  • 安装限制：尺寸、排水、电源、叠放方案。

空调 / 新风 / 净化器

  • 制冷制热速度：不同面积房间实测。
  • 温控舒适度：直吹、温差、湿度控制。
  • 能耗：不同模式耗电量、长期电费估算。
  • 噪音：低风、高风、睡眠模式。
  • 空气质量：颗粒物、甲醛、CO₂、换气效率。
  • 安装售后：打孔、铜管、支架、额外收费。

扫地机器人 / 洗地机

  • 清扫能力：灰尘、毛发、颗粒物、边角覆盖。
  • 拖地能力：顽固污渍、回洗、热风烘干。
  • 导航避障：桌椅、线材、宠物、地毯。
  • 基站体验：集尘、补水、排水、清洗维护。
  • App：地图、禁区、分区、多楼层、远程控制。
  • 耗材成本：尘袋、滤芯、清洁液、滚刷。

厨房小家电

  • 核心功能：破壁、空气炸、蒸烤、咖啡萃取等。
  • 实际出品：口感、熟度、均匀度、稳定性。
  • 操作难度：菜单、预设、老人友好度。
  • 清洗维护：拆洗、油污、异味、死角。
  • 安全：防烫、防溢、断电保护、材质安全。
  • 占地与收纳：厨房空间适配度。

智能门锁 / 安防设备

  • 解锁方式：指纹、人脸、密码、NFC、钥匙。
  • 识别速度与成功率：老人、儿童、湿手场景。
  • 安全性：锁芯等级、防撬报警、虚位密码。
  • 续航：电池容量、低电提醒、应急供电。
  • App：远程提醒、临时密码、家庭成员管理。
  • 安装适配：门体、天地钩、售后覆盖。

推荐结论模板

推荐等级 定义 适合表达 注意事项
S级 强烈推荐 同价位综合体验突出，缺点不影响核心体验。 闭眼买 / 同价位首选 / 大多数人都适合。 必须明确价格前提，避免绝对化。
A级 值得买 整体表现优秀，但有少量短板或适合特定人群。 如果你重视某项体验，这款很适合。 说明短板和替代选择。
B级 可考虑 有优势也有明显妥协，需看价格和需求。 降价后可以考虑 / 适合预算有限用户。 不要强推，强调前提条件。
C级 谨慎购买 核心体验存在明显不足，或价格竞争力弱。 除非特别喜欢这个卖点，否则不建议优先选。 给出更合理替代品。
D级 不推荐 体验、质量、售后或价格存在严重问题。 不建议买 / 建议避开。 结论要有证据，避免情绪化。

商单与带货合规清单

商业内容的核心不是“说好话”，而是“在真实前提下帮用户做选择”。长期信任是数码和家电内容账号最重要的资产。
✓ 视频中明确说明：自购、送测、借测或商业合作。
✓ 带货链接价格、优惠券、佣金性质不误导用户。
✓ 商单前确认品牌是否接受真实缺点呈现。
✓ 禁止虚构测试数据、虚构用户反馈。
✓ 竞品对比要基于事实，避免恶意贬低。
✓ 医疗、除菌、净化、节能等功效需谨慎表述。
✓ 推荐语避免“最好、第一、绝对、永久”等极限词。
✓ 评论区置顶补充价格变化、售后反馈、翻车信息。
