01

扫地机器人测评体系目标

对用户的价值

  • 帮用户判断：到底能不能解放双手。
  • 解释导航、避障、吸力、拖地、基站、耗材等复杂卖点。
  • 区分“看起来高级”和“长期用起来省心”。
  • 明确适合人群：宠物家庭、长发家庭、木地板、地毯、老人家庭。

对内容团队的价值

  • 固定测试场景，减少临时发挥和结论争议。
  • 形成可复用素材：清扫轨迹、避障、基站、App、噪音。
  • 建立横评数据库，方便大促推荐和品牌合作。
  • 提高商单议价能力和长期栏目化能力。

对商务合作的价值

  • 向品牌证明账号具备真实家居脏测能力。
  • 提前明确样品、耗材、售后、数据披露边界。
  • 把单次测评升级为新品测评、横评、大促转化、长期合作。
扫地机器人测评的核心不是跑一遍地图，而是模拟真实家庭长期清洁：灰尘、毛发、湿污、障碍、地毯、基站维护和 App 策略，最终回答“它能不能稳定帮我干活”。
02

测试场景搭建

场景 A：标准硬地清扫区

  • 用途：测试基础吸扫能力和覆盖率。
  • 地面：瓷砖、木地板、石塑地板任选其一，建议保留固定测试区。
  • 面积：建议 6-10㎡，可用胶带标出测试边界。
  • 脏污：灰尘、面粉、米粒、猫砂、咖啡粉、碎屑。
  • 测试重点：一次清洁率、边角残留、滚刷缠绕、吸口残留。

场景 B：拖地湿污测试区

  • 用途：测试拖地压力、拖布湿润度、顽固污渍处理。
  • 地面：建议瓷砖或防水硬地，方便反复清洁。
  • 污渍：酱油、咖啡、泥水、脚印、干掉的糖水、厨房油污模拟。
  • 测试重点：一次拖净率、水痕、回洗频率、二次污染。
  • 注意：同类污渍需控制用量和干燥时间。

场景 C：毛发与宠物家庭测试区

  • 用途：测试毛发拾取、防缠绕和宠物家庭适配。
  • 材料：长发、短发、宠物毛、纸屑、猫砂。
  • 布局：硬地 + 地毯 + 桌椅腿附近。
  • 测试重点：主刷缠绕、边刷缠绕、尘盒堵塞、猫砂飞溅。
  • 注意：毛发重量和长度要记录，横评时保持一致。

场景 D：复杂避障测试区

  • 用途：测试导航、避障、识别和脱困能力。
  • 障碍：数据线、袜子、拖鞋、玩具、宠物碗、纸巾、低矮门槛。
  • 家具：桌椅腿、沙发底、床底、餐桌区域。
  • 测试重点：是否碰撞、是否推走障碍、是否缠线、是否漏扫。
  • 注意：避障表现受光线影响，白天/夜间都建议测试。

场景 E：地毯与越障测试区

  • 用途：测试地毯识别、增压、抬拖布和越障。
  • 材料：短绒地毯、长绒地毯、地垫、门槛条。
  • 高度：建议准备 1cm、1.5cm、2cm 门槛或压条。
  • 测试重点：能否上地毯、是否打湿地毯、是否卡住。
  • 注意：不同拖布升降高度会明显影响结论。

场景 F：基站长期使用区

  • 用途：测试基站是否真的省心。
  • 条件：固定摆放位置，预留左右和前方空间。
  • 项目：自动集尘、自动洗拖布、热水洗、热风烘干、自动上下水。
  • 测试重点：回充成功率、拖布异味、污水箱、维护频率。
  • 注意：至少连续使用 7 天以上才有参考价值。

扫地机器人测试项目总清单

一、扫地能力测试

  • 硬地灰尘清洁率：面粉、咖啡粉、细灰。
  • 大颗粒清洁率：米粒、麦片、猫砂。
  • 边角清洁：墙边、桌腿、踢脚线附近残留。
  • 缝隙清洁：地板缝、瓷砖缝、门槛边。
  • 二次污染：是否扬尘、是否推着脏污走。
  • 尘盒残留：吸口、滤网、主刷仓是否堵塞。

二、拖地能力测试

  • 新鲜湿污：咖啡、酱油、泥水。
  • 干污渍：干掉的饮料、脚印、厨房油污模拟。
  • 拖布湿润度：是否过湿、是否拖不湿。
  • 水痕残留：木地板和瓷砖观感。
  • 沿边拖地：墙边、柜边是否能覆盖。
  • 回洗后拖布状态：是否越拖越脏。

三、毛发与防缠绕测试

  • 长发拾取：10cm、20cm、30cm 长发。
  • 宠物毛拾取：短毛、浮毛、毛团。
  • 主刷缠绕程度：测试后拆开观察。
  • 边刷缠绕程度：是否需要手动清理。
  • 尘盒堵塞：毛发是否堵住吸口或滤网。
  • 毛发切割 / 防缠结构是否有效。

四、导航建图测试

  • 首次建图速度和准确性。
  • 房间分区是否准确。
  • 家具腿、沙发底、床底覆盖情况。
  • 是否漏扫、重复扫、迷路。
  • 多楼层地图切换。
  • 断点续扫和回充成功率。

五、避障与脱困测试

  • 电线识别：是否缠绕或拖走。
  • 袜子、拖鞋、玩具识别。
  • 宠物便便模拟物识别。
  • 低矮家具进入与脱困。
  • 门槛、地垫、地毯边缘脱困。
  • 夜间避障表现。

六、地毯与越障测试

  • 短绒地毯清洁能力。
  • 长绒地毯是否缠绕或卡住。
  • 地毯识别是否准确。
  • 地毯自动增压是否明显。
  • 拖布抬升是否打湿地毯。
  • 1cm / 1.5cm / 2cm 越障能力。

七、基站功能测试

  • 自动集尘是否干净，集尘噪音多大。
  • 拖布自动清洗效果。
  • 热水洗拖布是否有效。
  • 热风烘干时间、异味、潮湿残留。
  • 自动上下水稳定性。
  • 污水箱、清水箱、尘袋维护频率。

八、App 与智能体验测试

  • 地图编辑、禁区、虚拟墙、房间命名。
  • 清洁策略：扫拖顺序、回洗间隔、水量、吸力。
  • 定时清洁、分区清洁、指哪扫哪。
  • 语音助手和智能家居联动。
  • 远程控制稳定性。
  • 老人和非数码用户是否容易上手。

扫地机器人测评评分模型

维度 权重 核心测试 结论表达
扫地清洁能力 20% 灰尘、大颗粒、猫砂、边角、缝隙、尘盒残留。 能不能把日常灰尘和颗粒扫干净。
拖地清洁能力 20% 湿污、干污、水痕、沿边、拖布回洗、二次污染。 能不能替代日常轻拖地。
导航与覆盖率 15% 建图、路径、漏扫、重复、回充、多楼层。 能不能稳定把家里扫完整。
避障与脱困 15% 电线、袜子、拖鞋、玩具、门槛、夜间避障。 会不会经常卡住、缠住、需要人工救援。
基站与维护 15% 集尘、洗拖布、烘干、上下水、污水箱、耗材。 是不是真的省心，还是换一种麻烦。
毛发 / 宠物家庭适配 5% 长发、宠物毛、猫砂、防缠绕、滤网堵塞。 长发和宠物家庭能不能放心买。
App 与智能体验 5% 地图、禁区、策略、语音、智能家居联动。 设置是否简单，日常是否好用。
价格与耗材成本 5% 到手价、历史低价、尘袋、拖布、清洁液、滤芯。 当前价格和长期使用成本是否合理。

  • 如有带货链接，说明可能存在佣金。

测试红线与商单红线

核心原则：扫地机器人属于强体验型家电，必须做真实脏测和长期体验。不能只拍品牌演示场景，更不能因为商单隐藏卡顿、缠绕、异味、漏扫、避障失败等关键问题。

测试红线

  • 禁止 虚构清洁率、避障成功率、续航、噪音等测试数据。
  • 禁止 用品牌提供的“理想演示物料”替代真实家庭脏污。
  • 禁止 横评时给不同机器设置不同清洁模式或脏污量。
  • 禁止 只测一次就下长期省心结论。
  • 禁止 隐藏缠毛、漏扫、拖布异味、基站维护麻烦等关键缺点。
  • 禁止 将工程机、特殊固件机表现当作零售机结论。

商单红线

  • 禁止 接受“不能拍失败画面”的合作。
  • 禁止 接受品牌要求删除真实缺点或修改推荐等级。
  • 禁止 未披露商业合作、样品来源或佣金关系。
  • 禁止 恶意贬低竞品或使用无依据排名。
  • 禁止 使用“彻底解放双手、永不缠毛、全屋零残留”等绝对化表述。
  • 禁止 未经授权声称官方合作或官方推荐。

可以接受的品牌修改

  • 允许 修改产品名称、型号、价格、参数等事实错误。
  • 允许 补充官方功能解释、固件计划和售后政策。
  • 允许 修改未上市信息、保密信息、发布时间。
  • 允许 指出测试方法中的技术错误并要求复测。

不能接受的品牌修改

  • 不允许 删除真实发生的卡住、缠线、漏扫、回充失败。
  • 不允许 要求把“不推荐”改成“推荐”。
  • 不允许 要求隐藏基站异味、污水箱维护、耗材成本。
  • 不允许 要求删掉样品来源和商业合作披露。

商务开发目标客户

  • 扫地机器人品牌：科沃斯、石头、追觅、云鲸、小米、米家、美的、海尔、iRobot 等。
  • 电商平台：京东、天猫、抖音电商、拼多多、苏宁易购。
  • 渠道商：品牌旗舰店、线上授权店、家电卖场。
  • 配件耗材：尘袋、拖布、清洁液、滤芯、边刷、主刷。
  • 家居清洁相关品牌：宠物用品、家政清洁、全屋智能。

首次商务沟通要问清楚

  • 推广目标：新品曝光、技术解释、口碑建设还是大促转化？

  • 审核范围：是否只审核事实错误，不干预结论。
  • 是否需要竞品排他、内容授权、二次投流。
  • 付款节点：预付比例、尾款时间、发票要求。

带货转化 SOP

发布前

  • 确认主推型号、上下水版、非上下水版和备选型号。
  • 确认京东 / 天猫 / 抖音等链接和佣金比例。
  • 记录当前价格、历史低价、大促预估价。
  • 准备不同人群购买清单：宠物、长发、地毯、大户型、小户型。
  • 确认是否可叠加国补、平台券、店铺券、以旧换新。

发布中

  • 简介区放购买链接和价格记录时间。
  • 评论区置顶推荐型号、适合人群、优惠说明。
  • 章节标注“购买建议”和“缺点避坑”。
  • 视频中说明价格会变化，以下单时页面为准。
  • 避免“全网最低”“永久保价”“彻底解放双手”等不可控表述。

发布后

  • 每日更新价格变化、优惠券和库存状态。
  • 回复用户户型、宠物、地毯、上下水安装问题。
  • 统计链接点击、成交、佣金、退货。
  • 收集售后、耗材、故障反馈并补充说明。
  • 大促后复盘哪些型号转化最好。
