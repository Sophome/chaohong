第一部分：电视选购的入门要点

时间 画面内容 旁白/台词 字幕/包装 音效/剪辑备注
01:15—01:28 电视开机广告、复杂首页、多个会员入口的模拟画面。 很多人现在不爱看电视，不是因为电视没用，而是因为广告多、操作麻烦、会员太复杂。 痛点：广告多 / 操作难 / 会员乱 模拟焦虑感。
01:28—01:45 遥控器特写，一键开机、一键进入B站或应用。 所以入门电视，第一优先级不是参数，而是两点：无开机广告，操作简单。 无广告 + 易操作 字幕重点放大。

02:00—02:18 老人使用语音搜索，儿童一键打开动画片。 对家里有老人、小孩的用户来说，一键直达、AI语音、老人模式，比很多纸面参数更重要。 老人模式 / AI语音 / 一键直达 家庭化场景。
02:18—02:30 电视首页显示主流视频APP、B站等应用入口。 电视好不好用，片源也很关键。现在很多电视可以直接安装主流视频APP。 片源便利性很重要 APP画面注意版权。
02:30—02:45 外接电视盒子、NAS、U盘、第三方应用的示意图。 也可以外接电视盒子，或者通过NAS、U盘、局域网播放本地资源。 电视盒子 / NAS / 本地播放 用示意动画即可。
02:45—03:00 主持人总结。 入门电视记住一句话：系统别烦人，操作别复杂，基础体验稳定，比堆参数更重要。 能用、好用、不烦人 小结卡片。

第二部分：电视选购的进阶要点

时间 画面内容 旁白/台词 字幕/包装 音效/剪辑备注
03:00—03:12 电视屏幕结构剖面动画：背光、液晶层、量子点膜。 接下来进入进阶部分。电视画质的基础，首先看面板和背光技术。 进阶关键词：面板技术 科技感动画。
03:12—03:28 参数条显示：面板成本约占整机成本40%—60%。 面板大约占到电视制造成本的40%—60%，直接决定亮度、色彩和对比度基础。 面板决定画质底线 数据来源需确认。
03:28—03:50 四类技术路线图：Mini LED → QD-Mini LED → SQD-Mini LED → RGB Mini LED。 现在主流路线包括Mini LED、QD-Mini LED、SQD-Mini LED和RGB Mini LED。 技术路线一图看懂 动画从低到高。
03:50—04:10 普通LCD和Mini LED亮暗对比画面。 Mini LED相比传统LCD，峰值亮度更高、控光能力更强，是入门到中端的主流升级方向。 Mini LED：入门升级 需实拍或授权对比画面。
04:10—04:28 量子点色彩增强示意。 QD-Mini LED通过量子点技术提升色彩表现，是中端性价比机型常见方案。 QD-Mini LED：中端主流 色彩渐变动画。
04:28—04:45 高端电视展示，高亮、高饱和HDR画面。 SQD-Mini LED可以理解为在量子点技术上进一步强化，通常出现在高端机型上。 SQD-Mini LED：高端方向 表述需谨慎。
04:45—05:05 RGB三色灯珠发光示意，去掉量子点膜。 RGB Mini LED则是另一条高端路线，采用红绿蓝三色灯珠发光，色彩和能耗表现更有潜力。 RGB Mini LED：旗舰路线 用三色光效动画。
05:05—05:18 四档选择建议卡片。 简单记：Mini LED看入门，QD-Mini LED看中端，SQD-Mini LED看高端，RGB Mini LED看旗舰。 选购速记 卡片保留3秒。
05:18—05:35 HDR高光画面：太阳、灯光、金属反光。 亮度也很重要。比如杜比视界内容，通常需要1000尼特峰值亮度打底。 峰值亮度：1000尼特起步更稳 数据图表。
05:35—05:50 分区控光示意：少分区 vs 多分区。 分区越多，控光越细，但分区也有边际效应，不是越多就一定越值得。 分区不是越多越好 网格动画。
05:50—06:05 主持人总结。 买电视不能只盯一个参数，要看亮度、分区、算法、面板方案的综合表现。 看综合，不看单一参数 小结卡片。
06:05—06:20 白天客厅，落地窗光线打到电视上，屏幕反光严重。 电视画质再好，如果白天一开全是反光，也会极大影响观看体验。 亮厅用户重点看抗反 真实场景拍摄。
06:20—06:38 三类抗反技术卡片：AG、LR/AR、AGLR。 目前常见抗反方案主要有三类：AG防眩光、LR或AR低反镀膜，以及AG+LR复合方案。 AG / LR-AR / AGLR 专业图解。
06:38—06:55 磨砂屏与低反屏对比：清晰度、通透度。 磨砂雾面可以减少镜面反射，但雾度过高会影响通透感和清晰度。 防眩光≠一定好画质 对比画面。
06:55—07:15 显示“AG25+LR低反膜”示意，旁边标注“高端配置”。 更好的方案是防眩光和低反射结合，既压反光，又尽量保持画面通透。 高端抗反：AG25 + LR 数据需与实测对应。
07:15—07:35 西晒客厅、落地窗、夜晚开灯三种场景。 如果你家是西晒客厅、落地窗正对电视，抗反能力要重点看；如果主要晚上看片，中高端低反方案通常已经够用。 极端采光才需要极致抗反 场景切换。
07:35—07:45 主持人出镜。 真要解决亮厅问题，有时候一副好窗帘，比盲目上旗舰更划算。 电视 + 窗帘一起考虑 轻松语气。

第三部分：电视选购的高端要点

08:00—08:18 高对比画面：墙面纹理、暗部细节、亮部层次。 第一是更精细的控光能力。同样亮度和分区，不同算法会带来完全不同的层次感。 A1 精细控光 需补实拍对比。
08:18—08:35 老片源修复前后：噪点、清晰度、人物面部细节。 第二是去噪、超分辨率解析和景深重构。老片源、低清片源会明显受益。 A2 去噪 / 超分 / 景深重构 前后对比。
08:35—08:52 人物肤色对比：偏红、偏黄、自然肤色。 第三是色彩还原，尤其是肤色。肤色一旦偏红或偏灰，会非常影响观感。 A3 肤色校准很重要 人像画面。
08:52—09:10 游戏画面、赛车、足球、动作片高速运动。 第四是刷新率和背光响应。高端电视通常支持4K 150Hz甚至更高刷新率。 A4 高刷新 + 快响应 需补游戏素材。
09:10—09:25 小球残影测试：普通背光 vs 高速背光。 同样刷新率下，背光响应越快，运动残影越少，游戏和体育画面更清晰。 残影越少，运动越清晰 需补测试画面。
09:25—09:35 色调自定义菜单画面。 第五是画质色调自定义，适合对电影、游戏、球赛有不同偏好的用户。 A5 色调自定义 原文A4重复，已修正。
09:35—09:50 电视内置扬声器结构图：左右声道、低音炮、天空声道。 很多人买电视只看画质，但高端电视的音质差异也很明显。 高端电视也要听声音 音波动画。
09:50—10:08 参数卡片：2.0、2.1、4.1.2、功率、低音单元。 入门电视通常是左右声道，高端电视可能做到4.1.2甚至更多声道，并支持杜比音效。 看声道、功率、低音配置 参数卡片。
10:08—10:25 电影爆炸、演唱会、人声对白三类场景。 好的电视音响，在人声、低频和空间感上都会更完整。 人声清楚 / 低频有力 / 声场更宽 声音设计突出。
10:25—10:40 声场校正动画：客厅反射、声波校准。 高端电视还会加入AI音效管理和声场校正，根据房间空间优化声音同步和定位。 声场校正：适配不同客厅 动画演示。
10:40—10:58 NAS、U盘、路由器网盘、电视影视墙界面。 中高端电视通常具备更强的视频解码和资源管理能力，比如影视墙、NAS播放、AI搜索。 解码能力影响本地观影体验 需补界面。
10:58—11:15 电视连接芝杜播放器、Apple TV、游戏主机。 但要注意，内置播放器适合偶尔看NAS电影，真发烧还是建议外接专业播放器。 发烧友建议：播放器外接 平衡表述。
11:15—11:32 PS5/Xbox连接电视，HDMI 2.1接口特写。 游戏玩家要重点看HDMI 2.1接口、4K高刷、VRR、ALLM等功能。 HDMI 2.1 / 4K高刷 / VRR 接口特写。
11:32—11:45 功放、回音壁、家庭影院连接图。 如果你要接回音壁、功放或家庭影院，还要看eARC音频回传。 影院扩展：eARC很重要 线路图动画。

  • 分区控光和肤色还原对比
  • 游戏高刷和残影测试
  • HDMI、eARC、NAS播放实拍

口播风格

  • 少讲术语，多讲场景
  • 少堆参数，多讲判断逻辑
  • 避免绝对化表述
  • 技术名词用一句话解释

精简版口播金句

  1. 入门电视先别看参数，先看有没有开机广告、系统卡不卡、老人会不会用。
  2. 中端电视重点看面板、亮度、分区和抗反，不要只盯一个参数。
  3. 高端电视的差距，不只在屏幕，更在芯片、算法、音质、游戏和扩展能力。
  4. 普通家庭优先75寸和85寸，大客厅有条件可以直接考虑100寸。
  5. 亮厅用户一定看抗反，游戏用户一定看HDMI 2.1和高刷，观影用户重点看控光、肤色和解码。
